# AutoFix Swarm - Branding Colors

## Severity Colors (Primary)
```css
--critical: #DC2626      /* Red - Security, crashes, critical bugs */
--warning: #F59E0B       /* Amber - Performance, deprecated code */
--caution: #EAB308       /* Yellow - Code smells, potential issues */
--info: #3B82F6          /* Blue - Suggestions, best practices */
--success: #10B981       /* Emerald - Fixed bugs, passing tests */
```

## Base Colors (Dark Theme)
```css
--background: #2D2D2D    /* Graphite - Main background */
--surface: #3A3A3A       /* Lighter graphite - Cards, panels */
--surface-elevated: #454545  /* Elevated elements */

--text-primary: #E6E1D7   /* Soft linen - Headings, primary text */
--text-secondary: #B8B3A8 /* Muted linen - Body text */
--text-muted: #8A8780    /* Hints, labels */

--border: #4A4A4A        /* Borders */
--accent: #D9A441        /* Honey bronze - Primary actions, links */
```

## Branding Guidelines

**Theme:** Autonomous bug detection & remediation
**Tone:** Professional, technical, trustworthy
**Visual Identity:** Clean, modern, warm-tech aesthetic

**Logo Color:** #F6AA1C (Golden Orange) - for maximum visibility on dark navbar and brand consistency

**Color Usage:**
- Use severity colors **only** for bug status indicators
- Dark graphite background (#2D2D2D) with warm linen text for professional dev tool feel
- Golden orange (#F6AA1C) for logo, CTAs, and primary interactions
- Warm palette (deep brown → golden orange gradient) for backgrounds and cards
- High contrast for accessibility (WCAG AA minimum)
